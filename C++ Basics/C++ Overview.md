##C++概览##

> C++ is a statically typed, compiled,
> general-purpose, case-sensitive,
> free-form programming language that
> supports procedural, object-oriented,
> and generic programming.

c++的特性：静态特性。编译性。通用性。变量、函数区分大小写。使用起来非常自由――支持过程编程，面向对象编程。泛型编程。

> C++ is regarded as a middle-level
> language, as it comprises a
> combination of both high-level and
> low-level language features.

c++是一种“中层”语言，结合了高级语言与低级语言的特性。

> starting in 1979 at Bell Labs in

> enhancement to the C language and
> originally named C with Classes but

##面向对象编程：##

> C++ fully supports object-oriented
> programming, including the four
> pillars of object-oriented
> development:

c++完全支持面向对象编程。包含面向对象的四大基本特性。

  - 1.Encapsulation――封装性
  - 2.Data hiding――数据隐藏（私有，公有）
  - 3.Inheritance――继承
  - 4.Polymorphism――多态性

##标准库##

C++标准库包含三个重要的部分：

1. 语言核心提供所有的构建模块。包括变量，数据类型和文字等。
2. C ++标准库中提供了一整套功能丰富的操作比如文件，字符串，等等。
3. 标准模板库（STL）提供了一整套丰富的数据结构极其操作。
##ANSI标准##

ANSI标准是为了确保C ++的可移植性 比如你在微软的平台上写了代码，那么同样的在其他平台上跑也没有问题

ANSI标准已经稳定了很长时间，基本上所有的厂商都认同ANSI标准。

学习一门编程语言的目的是为了成为一个更好的程序员;也就是说，要在设计和发布新系统的同时，保证代码能有效更新和维护。
